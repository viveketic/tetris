# Tetris
Classic tetris game in C++ using SDL2.0.

## Build
Make sure you have cmake and True Type Font (TTF) and SDL2-mixer installed.  

```
mkdir build
cd build
cmake -D CMAKE_BUILD_TYPE=Debug ../
make
```
## Run

Find binaries in `bin` directory.
```
./tetris 
```
